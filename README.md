# PythonForBeginners

A beginner friendly Python learning repository containing Jupyter notebooks for practicing Python concepts step by step.

---

```text
PythonForBeginners/
```

---

## goodies/

This folder contains helpful screenshots, visual references, and quick comparison charts to make Python concepts easier to understand and remember.  
These visuals are designed to support beginners with simple explanations and real world understanding of core Python topics.

---

## notebooks/

This folder contains step by step Jupyter notebooks covering Python concepts from basics to intermediate topics.  
Each notebook includes explanations, examples, and practice code designed to help beginners learn Python in a simple and hands on way.

---

## How to Execute the Notebooks

You can execute these notebooks using Google Colab without installing Python on your computer.

---

# Option 1: Open Notebook Directly from GitHub into Google Colab

This is the easiest method.

## Steps

### Step 1
Open the notebook in GitHub.

Example:

```text
https://github.com/deepakveeravalli/PythonLabs
```

---

### Step 2
Open the required notebook.

Example:

```text
01_Python_Introduction.ipynb
```

---

### Step 3
In the browser URL, replace:

```text
github.com
```

with:

```text
githubtocolab.com
```

Example:

FROM:

```text
https://github.com/deepakveeravalli/PythonLabs/blob/main/notebooks/01_Python_Introduction.ipynb
```

TO:

```text
https://githubtocolab.com/deepakveeravalli/PythonLabs/blob/main/notebooks/01_Python_Introduction.ipynb
```

---

### Step 4
Press Enter.

The notebook opens automatically in Google Colab.

---

### Step 5
Execute notebook cells using:

```text
Shift + Enter
```

or run the complete notebook using:

```text
Runtime → Run All
```

---

# Option 2: Open GitHub Repository from Google Colab

This method allows you to browse the repository directly from Colab.

## Steps

### Step 1
Open Google Colab:

https://colab.research.google.com

---

### Step 2
Click:

```text
File → Open notebook
```

---

### Step 3
Select the:

```text
GitHub
```

tab.

---

### Step 4
Paste the repository URL:

```text
https://github.com/deepakveeravalli/PythonLabs
```

---

### Step 5
Press Enter.

Colab displays all notebooks available in the repository.

---

### Step 6
Select the notebook you want to execute.

Example:

```text
01_Python_Introduction.ipynb
```

---

### Step 7
Run notebook cells using:

```text
Shift + Enter
```

or run the full notebook using:

```text
Runtime → Run All
```
